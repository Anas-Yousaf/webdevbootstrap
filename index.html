Bootstrap 5 Grid System

Responsive Layout Made Simple

Bootstrap's grid system is a powerful flexbox-based layout system that enables you to create responsive, mobile-first layouts with ease. It uses a series of containers, rows, and columns to layout and align content.

  • Mobile-First Approach: Designed for mobile devices first, then scales up
  • Flexbox Based: Built on CSS Flexbox for better alignment and distribution
  • 12-Column System: Divides the page into 12 equal columns
  • Responsive Breakpoints: 6 different screen size breakpoints

Container

The Foundation of Your Layout

Containers are the most basic layout element in Bootstrap and are required when using the grid system. They center and horizontally pad your content.

Types of Containers:

.container

Responsive fixed-width container with max-widths at each breakpoint

.container-fluid

Full-width container that spans the entire viewport width

.container-{breakpoint}

100% width until the specified breakpoint

<!-- Fixed-width container --> <div class="container"> Content here... </div> <!-- Full-width container --> <div class="container-fluid"> Content here... </div> <!-- Responsive container --> <div class="container-md"> Content here... </div>

Rows

Horizontal Groups of Columns

Rows are wrappers for columns. Each row creates a horizontal group of columns and must be placed within a container.

Key Features of Rows:

  • Flexbox Container: Uses display: flex for column alignment
  • Negative Margins: Counteract container padding
  • Gutters: Provide spacing between columns
  • Column Wrapping: Columns wrap to new line when total exceeds 12
<div class="container"> <div class="row"> <div class="col"> Column content </div> </div> </div>

Columns

The Building Blocks of Your Layout

Columns are the individual content areas within rows. Bootstrap's 12-column system allows for flexible layouts.

Column Types:

.col

Equal-width columns that automatically adjust

.col-{number}

Specific width columns (1-12)

.col-{breakpoint}-{number}

Responsive columns with breakpoint-specific widths

<!-- Equal width columns --> <div class="row"> <div class="col">1 of 3</div> <div class="col">2 of 3</div> <div class="col">3 of 3</div> </div> <!-- Specific width columns --> <div class="row"> <div class="col-sm-6">50%</div> <div class="col-sm-6">50%</div> </div>

Advanced Grid Features

Beyond Basic Layouts

Alignment Options:

  • Vertical Alignment: align-items-start, center, end
  • Horizontal Alignment: justify-content-start, center, end
  • Self Alignment: align-self-start, center, end
  • Auto Margins: ms-auto, me-auto for column positioning

Ordering:

  • Visual Order: order-1 through order-5
  • Responsive Ordering: order-sm-1, order-md-2, etc.
  • First/Last: order-first, order-last
<!-- Vertical alignment --> <div class="row align-items-center"> <div class="col">Centered content</div> </div> <!-- Horizontal alignment --> <div class="row justify-content-center"> <div class="col-6">Centered column</div> </div> <!-- Column ordering --> <div class="row"> <div class="col order-last">First in DOM, last visually</div> <div class="col order-first">Second in DOM, first visually</div> </div>

Best Practices

Tips for Effective Grid Usage

Do's:

  • Mobile-First: Start with mobile layout, then enhance for larger screens
  • Semantic HTML: Use appropriate HTML elements within grid structure
  • Consistent Gutters: Maintain consistent spacing throughout your design
  • Test Responsiveness: Always test on multiple screen sizes
  • Use Containers: Always wrap rows in containers

Don'ts:

  • Don't nest rows directly: Always wrap rows in containers or columns
  • Don't exceed 12 columns: Columns will wrap to next line
  • Don't skip breakpoints: Consider all screen sizes
  • Don't ignore accessibility: Ensure proper heading hierarchy
  • Don't over-complicate: Keep layouts simple and maintainable
